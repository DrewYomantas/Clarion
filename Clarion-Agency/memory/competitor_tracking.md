# competitor_tracking.md
# Clarion Agent Office — Competitor Intelligence Log
# Version: 1.0 | Created: 2026-03-12
#
# PURPOSE
# Append-only log of discrete competitor observations: product moves, messaging shifts,
# pricing signals, new launches, and positioning changes.
# This is the timestamped evidence trail behind Clarion's competitive view.
# It feeds the Market Intelligence agent's weekly synthesis and the Chief of Staff's
# COMPETITOR INTELLIGENCE SUMMARY in the CEO brief.
#
# WHO WRITES TO THIS FILE
# Competitive Intelligence Analyst — appends an entry whenever a discrete, verifiable
# signal is observed about a tracked competitor or watchlist company.
# Market Intelligence Agent — may also append entries from strategic synthesis runs.
#
# RULES
# - Append-only. Never edit or delete entries.
# - One entry per discrete observation. Do not bundle multiple signals into one entry.
# - Write entries at the time of observation. Do not backfill.
# - OBSERVATION must describe exactly what was seen — not an interpretation.
#   Interpretation belongs in POTENTIAL_IMPACT.
# - SOURCE must be a specific, attributable public source (URL, platform, publication).
#   Do not write "general research" or "various sources."
# - POTENTIAL_IMPACT is the agent's honest assessment. One sentence. It is an
#   observation, not a directive. Product or positioning decisions require CEO review.
# - IMPACT_LEVEL values: Low | Medium | High
#   High = direct threat to Clarion's core positioning, ICP, or active pipeline
#   Medium = relevant shift worth tracking; no immediate action required
#   Low = informational; peripheral or unlikely to affect Clarion
# - Do not log unverified rumors. If a signal is uncertain, note "Unverified" in SOURCE.
#
# OBSERVATION_TYPE values:
# product_launch       — new product, feature, or capability announced or released
# pricing_change       — publicly visible pricing restructure, new tier, discount
# messaging_shift      — new positioning, tagline, category claim, or audience targeting
# partnership          — announced integration, channel deal, or co-marketing
# funding              — funding round, acquisition, or financial event
# customer_sentiment   — pattern in public reviews or community feedback about a competitor
# executive_move       — leadership hire, departure, or public statement
# market_entry         — competitor entering law firm or adjacent vertical explicitly
# watchlist_signal     — activity from a watchlist company not yet fully profiled
# other                — use when none of the above apply; describe in OBSERVATION
#
# Market Intelligence Agent reads this file weekly.
# Chief of Staff reads this file weekly via the Market Intelligence Agent report.
# See agents/strategy/market_intelligence.md and agents/executive/chief_of_staff.md.
