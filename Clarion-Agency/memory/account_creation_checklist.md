# account_creation_checklist.md
# Clarion — Pre-Account-Creation Checklist
# Agents must confirm all items before creating any external account.

---

## Required Checks (all must pass)

1. Platform appears in memory/channel_strategy.md under Primary or Secondary channels
   If not listed: do not create. Propose it to CEO first.

2. Platform does NOT already have an entry in memory/platform_registry.md
   If already registered: do not create a duplicate. Update the existing entry instead.

3. ICP plausibly exists on this platform in a buying or research context
   Managing partners or legal ops professionals must be reachable there.

4. Purpose is defined and specific
   "Build presence" is not sufficient. State: what will be posted, how often, to what end.

5. Naming follows brand rules
   Handle must use "clarion" or "clarionhq" consistently.
   No personal names. No alternate spellings.

6. CEO approval exists in memory/approved_actions.md with STATUS: approved
   No account may be created without a matching approved action entry.

---

## After Creation

- Add entry to memory/platform_registry.md immediately
- Store credentials in the designated secure location (not in any memory file)
- File a brief WORK COMPLETED note in the next agent report
