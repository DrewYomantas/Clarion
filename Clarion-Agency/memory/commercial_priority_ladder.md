# commercial_priority_ladder.md
# Clarion — Agent Decision Priority
# When multiple work options exist, agents choose the higher-priority item.

---

## Priority Order

1. Work that can create revenue
   Examples: outreach angle development, demo conversion assets, pricing clarity work,
   early adopter targeting, messaging that reduces sales friction

2. Work that can create market credibility
   Examples: authority content for ICP, community participation that builds presence,
   competitive positioning that establishes a clear wedge

3. Work that can improve conversion
   Examples: landing page clarity, onboarding friction reduction, trial-to-paid path work,
   messaging that removes objections

4. Work that can improve product clarity
   Examples: feature explanation, documentation, governance brief quality

5. Work that supports later systems
   Examples: process documentation, templates, infrastructure that will matter post-launch

---

## How to Apply This

When an agent has two valid proposals and must choose one, choose the higher number.

When proposing actions to the CEO, note which priority tier the action falls in.

Do not propose Tier 4 or Tier 5 work as urgent if Tier 1 or Tier 2 work is unaddressed.
