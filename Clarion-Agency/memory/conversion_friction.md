# conversion_friction.md
# Clarion Agent Office — Conversion Friction Log
# Version: 1.0 | 2026-03-12
#
# PURPOSE
# Captures real reasons prospects hesitate, decline pilots, or fail to convert.
# This is the ground-truth record of where the sales process breaks down.
# It feeds the Chief of Staff weekly friction pattern review and informs
# product, messaging, and pricing decisions.
#
# WHO WRITES TO THIS FILE
# Outbound Sales Agent — appends an entry whenever a prospect:
#   - Declines a pilot
#   - Stops replying after outreach (ghosted after 2+ touchpoints)
#   - Expresses explicit hesitation (price, timing, relevance, trust, complexity)
#   - Asks a question that reveals a product gap or misunderstanding
#   - Closes lost at any stage
#
# RULES
# - Append-only. Never edit or delete entries.
# - Write entries at the time of the signal. Do not batch-log weeks later.
# - Be specific. Vague entries ("not interested") are not useful.
#   Capture the actual words, objection, or behavior if known.
# - Do not include named individuals or personally identifiable information.
#   Use firm type, geography, and practice area only.
# - If the prospect's exact words are known, quote them under EVIDENCE.
#   If not known exactly, paraphrase and note it as a paraphrase.
# - POTENTIAL_PRODUCT_IMPACT is the agent's observation only.
#   It is not a directive. Product decisions require CEO review.
#
# FRICTION TYPE VALUES
# price_sensitivity      — cost is the stated or implied objection
# timing                 — not the right time (budget cycle, hiring, caseload, etc.)
# relevance_doubt        — prospect does not believe the product applies to them
# trust_gap              — prospect is skeptical of value, AI, accuracy, or Clarion legitimacy
# process_friction       — prospect found onboarding, pilot setup, or workflow too complex
# no_decision_maker      — wrong contact; actual buyer is someone else
# competing_priority     — something else is consuming leadership attention right now
# product_gap            — Clarion lacks a feature or capability the prospect needs
# ghosted                — no reason given; stopped responding after 2+ touchpoints
# other                  — use when none of the above apply; describe in SUMMARY
#
# CHIEF OF STAFF reads this file weekly.
# See agents/executive/chief_of_staff.md — CONVERSION FRICTION REVIEW section.
