## Rules

Before creating any account:
1. Check this file — does the platform already have an entry?
2. Check memory/account_creation_checklist.md — confirm all conditions met
3. Get CEO approval via memory/approved_actions.md
4. After creation: add a row to this registry immediately

Status values: active | inactive | suspended | pending-setup

Credentials location: Never store credentials here. Note where they are stored (e.g., "1Password vault — Clarion Social").
